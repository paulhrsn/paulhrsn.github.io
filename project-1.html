SecVault

A browser-based file vault that encrypts files client-side with AES-256 before upload. Your files stay your files—end-to-end.

Live Link

Project Overview

SecVault is a secure file vault web app built for privacy-first users. It uses WebCrypto to encrypt files directly in the browser before they ever leave the user's device. The server never sees the unencrypted data, and files can be safely stored or shared with confidence.

It was built to demonstrate end-to-end encryption principles, client-server coordination, and secure frontend UX in a full-stack deployment.
